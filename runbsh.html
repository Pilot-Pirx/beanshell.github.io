BeanShell Applet

Note: bsh requires a Java 1.1 compliant web browser such as HotJava 1.0 or 1.1, Netscape Navigator 4.03 or 4.04 with the additional final 1.1 support patch, or the JDK 1.1.x appletviewer. I haven't seen the applet work with Netscape 4.05, MSIE 4.0+, or the JDK1.2 betas.

Remember that as an applet bsh is limited by the applet security model. For full access to system resources you should download bsh and run it stand-alone.

Tip: You can get a console window in an external frame at any time by typing: console() at the prompt. Use the editor() command for an editor window.
