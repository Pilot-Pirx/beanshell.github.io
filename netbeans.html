BeanShell in NetBeans/Forte

Upgrading BeanShell in NetBeans/Forte

If you use BeanShell in the NetBeans IDE and wish to upgrade the version of BeanShell, you simply have to replace the bsh jar file in the modules\ext directory with the new version. e.g.

  // NetBeans
  c:\netbeans\modules\ext\bsh-1.01.jar

  // Forte
  c:\forte4j/lib/ext/bsh-1_0-fj.jar

The only catch is that may have to rename the new jar file to the "bsh-1.01.jar" name, because it is hard-coded in the modules\scripting.jar manifest classpath. I hope they will make this more flexible in the future.
