  • Use the object() method to get an empty bsh object for holding data items. e.g.
    •   myStuff = object();
  myStuff.foo = 42;
  myStuff.bar = "blah";
  • If you'd prefer to work in your own text editor (rather than a bsh editor window) simply pop up an editor window with a source command pointing to your file: e.g. 
    •   source("/home/pat/myscript.bsh");
    Then you can simply hit the "eval" button each time you want to run it.
  • You can use #!/bin/java bsh.Interpreter to make executable bsh scripts in unix-like environments.
  • Test for undefined vars with == void, undefine a var with the unset("name") command.
